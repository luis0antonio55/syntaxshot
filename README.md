# SyntaxShot — Product Landing

This repository contains the product landing site and example React components for SyntaxShot — a CLI tool that generates syntax-highlighted screenshots from the terminal.

## Overview

The site is built with Vite + React and contains a landing page, docs, and reusable UI components.

## Requirements

- Node.js 18+ (recommended)
- npm or pnpm

## Install

Install dependencies:

```bash
npm install
# or with pnpm
pnpm install
```

## Development

Run the dev server:

```bash
npm run dev
# or
pnpm dev
```

Open the URL printed by Vite (usually http://localhost:5173).

## Project structure 

- `src/` — application source and pages
- `src/app/pages/Landing.tsx` — landing page
- `src/app/pages/Docs.tsx` — documentation page
